# Microsoft Software and Systems Academy Projects

### This repo is a small selection of projects and topics I trained on during my participation in MSSA.

- **99bottles**: Console app that prints the "99 Bottles of Beer" lyrics.

- **AiFoundryModelTextToSpeech**: Azure OpenAI chat prompt with speech synthesis output using user-secrets.

- **Interfaces**: OOP interfaces demo that cycles through gadgets and activates them.

- **IntergalacticPizza**: Console pizza ordering with planet delivery fees and discounts.

- **MortgageLoanCalculator**: Mortgage calculator console app with shared logic and tests.

- **ReverseChallenge**: Tiny console app that reverses user input.
